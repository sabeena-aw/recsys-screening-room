# Screening Room — Movie Recommender Prototype

A recommender systems prototype built on MovieLens `ml-latest-small`: six models (two
non-personalized baselines, content-based filtering, item-item and user-user collaborative
filtering, and matrix factorization via SGD), a shared evaluation harness, and a Streamlit app
to compare their recommendations side by side.

Built for the Recommender Systems individual project (Esade, Prof. Marc Torrens).

## Project structure

```
recsys/
  main.py              # runs the full pipeline: train all models, evaluate, save results
  eda.py                # generates the EDA figures used in the report/slides
  sanity_check.py        # verification script -- run before trusting any results
  app.py                  # Streamlit app
  src/
    config.py              # paths and constants
    data_loading.py          # load/validate ratings & items, EDA stats, train/test split
    baselines.py               # Most Popular, Highest Average, Random
    content_based.py             # TF-IDF over genres + tags
    collaborative_filtering.py     # item-item and user-user CF
    matrix_factorization.py          # SGD with bias terms
    evaluation.py                      # Precision/Recall/NDCG/Hit-Rate/Coverage/Novelty
  data/raw/                              # ml-latest-small CSVs (see Data below)
  results/                                 # metrics.csv + figures/, generated by main.py / eda.py
```

## Setup

```bash
pip install -r requirements.txt
python main.py            # trains everything, writes results/metrics.csv
python eda.py              # writes results/figures/*.png
python sanity_check.py      # verifies the implementation against independent calculations
streamlit run app.py         # interactive comparison UI
```

## Data

[MovieLens `ml-latest-small`](https://grouplens.org/datasets/movielens/latest/) — 100,836
ratings across 9,742 movies from 610 users, collected by GroupLens Research, University of
Minnesota. Used here under the dataset's research-use license (see `data/raw/README.txt`):
non-commercial use, with this acknowledgment in place of formal publication.

> F. Maxwell Harper and Joseph A. Konstan. 2015. The MovieLens Datasets: History and Context.
> ACM Transactions on Interactive Intelligent Systems (TiiS) 5, 4: 19:1–19:19.

## Method summary

| Model | Approach |
|---|---|
| Most Popular | Ranks by raw rating count |
| Highest Average | Ranks by mean rating, min. 20 ratings |
| Content-Based | TF-IDF over genres + tags, cosine similarity to a centered-rating user profile |
| Item-Item CF | Cosine similarity over the user-item matrix, top-k neighbor weighting |
| User-User CF | Same idea, transposed |
| Matrix Factorization | SGD on `r̂ = μ + b_u + b_i + p_u·q_i`, 30 factors, 15 epochs |

Full writeup, evaluation protocol, results, and limitations are in the accompanying slide deck.

## Verification

`sanity_check.py` cross-checks the baselines against raw pandas calculations done independently
of the model classes, confirms matrix factorization beats a trivial global-mean baseline on
held-out RMSE, and validates the evaluation metrics against a hand-computed example. Run it
after any change to confirm nothing broke.
